Challenge: Reverse a String

Write a function that takes a string as input and returns the string reversed.

Example 1

Input: "hello"

Output: "olleh"

Example 2

Input: "OpenAI"

Output: "IAnepO"

Hints

  • You can use the split, reverse, and join methods in JavaScript.
  • Remember that strings are immutable in JavaScript, so you need to convert it to an array first.
Editor
// Write your code here function reverseString(str) { return str.split("").reverse().join(""); } console.log(reverseString("hello")); console.log(reverseString("OpenAI"));
Console
> olleh > IAnepO
